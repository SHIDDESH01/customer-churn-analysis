# customer-churn-analysis

Customer Churn Analysis Dashboard
📌 Project Overview
Customer churn is one of the biggest challenges for subscription-based businesses. This project analyzes customer data to identify patterns and factors that lead to customer churn.
The dashboard was built using Power BI to visualize customer behavior, contract types, payment methods, and service usage to understand why customers leave.

🎯 Project Objectives
-Analyze customer churn patterns
-Identify high-risk customer segments
-Understand the impact of contract type and payment methods
-Analyze monthly charges and tenure distribution
-Help businesses reduce churn using data-driven insights.

Tools & Technologies Used
📊 Power BI – Data visualization and dashboard creation
🧹 Excel / Data Cleaning – Data preprocessing
🧮 DAX (Data Analysis Expressions) – Calculated measures
📂 Dataset – Telecom customer churn dataset
